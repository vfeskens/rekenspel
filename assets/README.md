# Assets-map

## Eigen muziekbestand gebruiken

Wil je een echt bestaand liedje als achtergrondmuziek in plaats van de ingebouwde melodie? Zet een **mp3**-bestand in deze map met de naam:

```
muziek.mp3
```

Zodra het bestand er staat, wordt het automatisch geladen bij het starten van het spel (pagina verversen). Het procedurele melodietje wordt dan uitgezet.

Heb je een ander formaat (m4a, wav, ogg)? Gebruik een gratis converter (bijv. ffmpeg of een online tool) om 'm naar mp3 om te zetten.

Let op: alleen bestanden die je legaal bezit gebruiken. Voor eigen gebruik thuis is dat prima.
